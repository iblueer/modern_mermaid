# 截图占位符 / Screenshot Placeholders

请将以下截图添加到此目录中：

## 必需的截图 / Required Screenshots

### 1. `banner.png`
- **用途**: 项目横幅图片
- **建议尺寸**: 1200 x 300 像素
- **内容**: 项目 logo 和标语

### 2. `screenshot-main.png`
- **用途**: 主界面截图
- **建议尺寸**: 1920 x 1080 像素
- **内容**: 显示完整的编辑器和预览界面

### 3. `screenshot-themes.png`
- **用途**: 主题展示
- **建议尺寸**: 1920 x 1080 像素
- **内容**: 多个主题的对比展示

### 4. `screenshot-annotations.png`
- **用途**: 标注工具展示
- **建议尺寸**: 1920 x 1080 像素
- **内容**: 使用标注工具的界面

### 5. `screenshot-export.png`
- **用途**: 导出选项展示
- **建议尺寸**: 1920 x 1080 像素
- **内容**: 导出菜单和选项

## 截图要求 / Screenshot Requirements

- **格式**: PNG（推荐）或 JPG
- **质量**: 高分辨率，清晰可见
- **内容**: 使用有意义的示例图表
- **主题**: 使用美观的主题配色
- **尺寸**: 建议使用 2x 或 3x 分辨率以支持高 DPI 显示器

## 截图工具建议 / Recommended Tools

- macOS: Cmd + Shift + 4
- Windows: Windows + Shift + S
- Linux: Flameshot / GNOME Screenshot
- 浏览器: 开发者工具截图功能
